# Parrot Browser on Python

Parrot Browser is coded in Python + Qt5

![Screenshot](screenshot.png?raw=true)

## Installation

Use the requirements.txt file

```bash
pip install -r requirements.txt
```
or 

```bash
pip3 install -r requirements.txt
```

## Usage

```bash
python main.py
```
or 

```bash
python3 main.py
```
